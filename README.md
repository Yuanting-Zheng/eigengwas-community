# EigenGWAS Community

Professor Guo-Bo Chen Research Group

## About

EigenGWAS Community focuses on population genomic selection scanning and bio-computing analysis, providing open-source tools and academic exchange platforms.

## Core Publications

- **EigenGWAS** (Mol Ecol Resour, 2021): An online visualizing and interactive application for detecting genomic signatures of natural selection - [PubMed](https://pubmed.ncbi.nlm.nih.gov/33665976/)
- **UK BioCoin** (Nature Communications): Swift trait-specific summary statistics regression for UK Biobank - [PubMed](https://pubmed.ncbi.nlm.nih.gov/41997943/)
